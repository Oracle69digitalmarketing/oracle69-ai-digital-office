# Oracle69 AI Digital Office Prompt Library

## Purpose

This directory contains all prompts used by the Oracle69 AI Digital Office.

Every AI employee is driven by one or more prompt files.

The prompts define:

- Identity
- Responsibilities
- Decision boundaries
- Communication style
- Memory usage
- Escalation rules
- Tool usage
- Output format

## Principles

Every prompt should:

- stay focused on one responsibility
- never perform another agent's work
- produce structured outputs
- use company memory
- follow Oracle69 operating procedures
- escalate uncertainty instead of guessing

The Chief of Staff coordinates all agents.

The CEO approves strategic decisions.

The Execution Engine manages workflow.

The Memory System provides organizational context.
